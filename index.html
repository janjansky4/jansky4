Úvod

Vítejte na stránce věnované krásám matematiky. Najdete zde důležité vzorce, vysvětlení a tipy.

Základní rovnice

Kvadratická rovnice:
\\[ ax^2 + bx + c = 0 \\]
Vzorec pro výpočet diskriminantu:
\\[ D = b^2 - 4ac \\]
Vzorec pro výpočet řešení:
\\[ x = \\frac{-b \\pm \\sqrt{D}}{2a} \\]

Teorie limit

Definice limity podle Weierstrasse:

\\[ \\lim_{x \\to a} f(x) = L \\iff \\forall \\varepsilon > 0\\; \\exists \\delta > 0:\\; 0 < |x - a| < \\delta \\Rightarrow |f(x) - L| < \\varepsilon \\]

Derivace a integrály

Derivace funkce: \\[ f'(x) = \\lim_{h \\to 0} \\frac{f(x+h) - f(x)}{h} \\]
Určitý integrál: \\[ \\int_a^b f(x)\\,dx \\]
